# Sports Selection Frontend Application

A beautiful, modern front-end application that allows users to select and follow their favorite sports teams and countries.

## Features

- **Landing Page**: Three visually stunning sport boxes (Football, Cricket, F1) with smooth animations
- **Selection Page**: Dynamic country/team selection interface based on chosen sport
- **Beautiful Design**: Modern gradient backgrounds, glassmorphism effects, and smooth transitions
- **Responsive**: Fully responsive design that works on all devices
- **Mock API**: Simulated API calls for saving user selections

## Project Structure

```
Project_1/
├── index.html              # Main landing page with sport boxes
├── selection.html          # Selection page for countries/teams
├── css/
│   ├── style.css          # Main stylesheet with beautiful design
│   └── selection.css      # Styles for selection page
├── js/
│   ├── main.js            # Landing page logic and navigation
│   ├── selection.js       # Selection page logic and API calls
│   └── api.js             # Mock API for saving selections
└── README.md              # Project documentation
```

## How to Use

1. Open `index.html` in a web browser
2. Click on any of the three sport boxes (Football, Cricket, or F1)
3. On the selection page, click on countries/teams to select them
4. Click "Save Selections" to save your choices
5. Use the "Back" button to return to the landing page

## Technologies Used

- HTML5
- CSS3 (with animations, gradients, and modern effects)
- Vanilla JavaScript (no frameworks)
- Open source images from Unsplash

## Browser Support

Works on all modern browsers:
- Chrome
- Firefox
- Safari
- Edge

## Design Features

- Animated gradient backgrounds
- Smooth hover effects and transitions
- Card-based UI with glassmorphism
- Responsive grid layouts
- Loading states and feedback messages
- Keyboard accessibility support

## Notes

- Images are loaded from Unsplash CDN
- Selections are saved to browser localStorage (simulating API calls)
- All animations and transitions are CSS-based for optimal performance
